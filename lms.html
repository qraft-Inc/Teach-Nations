Integration & LTI notes

  • Recommended: integrate via LTI 1.3 to securely launch courses from the main site and pass user context.
  • For LTI setup you will need: issuer URL, client ID, deployment ID, and a redirect URI on this site (e.g., https://your-site.example.com/lti/launch).
  • Alternatively, create direct enroll keys on Moodle and share course links with students.

Need help configuring Moodle or LTI? I can add example LTI config files and a simple serverless endpoint to handle OIDC token exchange.
